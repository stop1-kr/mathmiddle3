삼각함수 값의 변화 관찰

🤔 왜 선분의 길이가 삼각비의 값이 될까요?

중학교 3학년 수학 시간에 배우는 직각삼각형에서 삼각비의 정의는 다음과 같습니다.

  • sin x = (높이) / (빗변)
  • cos x = (밑변) / (빗변)
  • tan x = (높이) / (밑변)

1. sin x 와 cos x의 결정 (작은 직각삼각형 OAP)

좌측 사분원은 반지름의 길이가 1입니다. 즉, 원점 O에서 원 위의 점 P를 이은 선분 OP의 길이(빗변)는 항상 1이 됩니다.
점 P에서 x축에 수선의 발 A를 내린 직각삼각형 OAP에서 살펴보면:
- sin x = AP / OP = AP / 1 = 선분 AP의 길이 (빨간색 선)
- cos x = OA / OP = OA / 1 = 선분 OA의 길이 (파란색 선)

2. tan x의 결정 (큰 직각삼각형 OBC)

tan x를 구하기 위해 이번에는 밑변의 길이가 1인 더 큰 직각삼각형을 찾아봅니다. x축 위의 점 B(1, 0)에서 위로 수직선을 긋고, 선분 OP의 연장선이 만나는 점을 C라고 하면 직각삼각형 OBC가 만들어집니다.
이 삼각형의 밑변 OB의 길이는 사분원의 반지름이므로 1입니다.
- tan x = BC / OB = BC / 1 = 선분 BC의 길이 (초록색 선)

두 직각삼각형(△OAP, △OBC)은 서로 닮음(AA 닮음)입니다. 따라서 빗변이 1인 삼각형을 이용하면 sin, cos을 선분으로 나타낼 수 있고, 밑변이 1인 삼각형을 이용하면 tan를 직관적인 선분의 길이로 나타낼 수 있습니다. 각도(x)를 조절하며 우측 그래프에서 각 값이 어떻게 변하는지 확인해 보세요!
